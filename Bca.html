Bachelor of Computer Application (B.C.A)

Course Duration : 3 Years (Six Semesters)

The BCA Course aims at inculcating essential skills as demanded by the global software industry through interactive learning process. This also includes team-building skills, audio- visual presentations and personality development programmes. The programme enhances analytical and communication skill besides inculcating the virtues of self-study. The programme is divided into six semesters with two semesters in an academic year. Each semester has a working duration of about 21 weeks as per the following breakup.

  • The study period of BCA includes various subjects such as Programming in C Language (Basic and Advanced), Networking, World-Wide-Web, Data Structure, Advanced C Language Programming, Database Management, Mathematics, Software Engineering, Object Oriented Programming using C++, Visual Basic, Programming using PHP, JAVA, Oracle, Operating Systems, Web Scripting and Development, etc
  • Semester End Examination (including Practical) : 3 weeks

There is a break of 2 weeks after every odd semester and vacation of approximately 6 weeks after even semester.
